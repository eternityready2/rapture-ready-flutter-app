## :book: Features

* Firebase Push Notifications
* Firebase In-App Messasing
* UI (Layout + Content + Images) Generated using CMS
* UI Cached in case of Network Failure.
* In App Reviews
* Cross Platform (IOS and Android)

## :warning: Requirements

* Android 5.0+
* IOS 13.0+
